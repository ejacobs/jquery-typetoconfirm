Basic Usage

$('#yourButton').typeToConfirm({
    wordtotype: 'confirm'
    /* other options here */
});

Once instantiated, typeToConfirm will defer the onclick code until after confirmation. If the element is a form submission it will defer submission of the form until after confirmation (and and onclick event as well).

Note that any code applied to the element via bind() or click() will be executed as soon as the user clicks, before any confirmation.
